컨트롤러 버튼 사용 유무 변경하기

슬라이드의 버튼를 사용하거나, 사용하지 않도록 변경할 수 있습니다.
useButton : true 을 false 로 하면, 모든 컨트롤러의 버튼을 사용할 수 없습니다.
usedButtonNext : true 을 false 로 하면 다음 버튼을 사용할 수 없습니다.
usedButtonPrev : true 을 false 로 하면 이전 버튼을 사용할 수 없습니다.
usedButtonPause : true 을 false 로 하면 일시 정지 버튼을 사용할 수 없습니다.
usedButtonResume : true 을 false 로 하면 재생 버튼을 사용할 수 없습니다.

위 속성을 이용해서 버튼의 사용 유무를 제어할 수도 있고, 컨트롤러에서 필요한 버튼 코딩을 삭제함으로써 사용 유무를 제어할 수도 있습니다. 

                    $(".s3").imageSlider({
 usedButton : true,
 usedButtonNext : false,
 usedButtonPrev : false,
 usedButtonPause : true,
 usedButtonResume : true
});
