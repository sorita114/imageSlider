자동 슬라이드 제어하기

자동으로 슬라이드가 되는 것을 제어할 수 있습니다.
autoSlide : true 을 false 로 하면, next 나 prev 버튼을 클릭하지 않는 이상 자동으로 슬라이드가 되지 않습니다. 

                    $(".s4").imageSlider({
 autoSlide : false
});
